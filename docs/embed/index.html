Embed

You have several possibilities:
  • Create and publish a new tag on Google Tag Manager.
  • Send the generated snippet to your webmaster or a developer.
  • Add a block in your CMS and make sure it's visible on pages where you want the widget on.
If something doesn't work:
  • Make sure $pr isn't clashing with other JavaScript variables on your website.
  • Make sure the domain name (publisher id) is correct, embedding the snippet on websites with unrecognised publisher id will not work.
